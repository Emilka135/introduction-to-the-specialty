# 🔐 Password Manager

Современный менеджер паролей с генератором надёжных паролей и безопасным хранилищем.

## 📋 Описание проекта

**Password Manager** — это веб-приложение для генерации криптографически стойких паролей и их безопасного хранения. Приложение помогает пользователям создавать уникальные сложные пароли для разных сервисов и хранить их в зашифрованном виде.

### 🎯 Цель проекта
Создать удобный инструмент для управления паролями с интуитивным интерфейсом и надёжной защитой данных.

### ✨ Основные функции

#### Генератор паролей
- Настройка длины пароля (4-64 символа)
- Выбор типов символов:
  - Заглавные буквы (A-Z)
  - Строчные буквы (a-z)
  - Цифры (0-9)
  - Специальные символы (!@#$%^&*)
- Визуальная индикация надёжности пароля
- Копирование в один клик

#### Хранилище паролей
- Сохранение паролей с названием и описанием
- Шифрование паролей перед сохранением (bcrypt)
- Просмотр списка сохранённых паролей
- Быстрое копирование пароля
- Удаление ненужных записей

## 🛠 Технологический стек

### Frontend
- **HTML5** — семантическая разметка
- **CSS3** — современный дизайн с градиентами и анимациями
- **Vanilla JavaScript** — чистый JS без фреймворков
- **Fetch API** — работа с сервером

### Backend
- **Node.js** — среда выполнения
- **Express.js** — веб-фреймворк
- **bcryptjs** — хеширование паролей
- **CORS** — поддержка кросс-доменных запросов
